# 작성 규칙

## 설정 일관성
- 인물의 이름, 외형, 말투는 인물 파일 기준으로 고정
- 세계관 고유 용어는 용어집.md 기준으로 통일
- 능력/아이템 효과는 능력-체계.md 범위 내에서만 사용

## 복선 관리
- 새 복선을 심으면 → `소재/복선-목록.md`에 추가
- 복선을 회수하면 → `플롯/복선-회수표.md`에 표시

## 에피소드 작성 후 처리
- 완성된 화는 `요약/`에 한 줄 요약 추가
- 새 정보(인물/세계관 추가)는 해당 파일에 반영
- `context/current.md` 업데이트

## 커밋 규칙
- feat: 새 화 완성
- update: 기존 설정 수정
- fix: 오류/모순 수정
- plan: 플롯/구성 변경
