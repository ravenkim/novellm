# 정보 공개 흐름 (Revelation Plan)

> 독자에게 언제, 무엇을, 어떻게 공개할지 설계.
> "언제 모르게 할 것인가"와 "언제 알게 할 것인가"가 서사의 긴장을 만든다.

## 공개 흐름표
| 정보 ID | 내용 요약 | 공개 방식 | 공개 화 | 공개 대상 | 독자 감정 의도 |
|---------|---------|---------|--------|---------|-------------|
| T001    |         | 직접묘사/대화/암시 | | 독자만/인물도 | 긴장/충격/안도 |

## 공개 방식 유형
- **직접 묘사** — 서술자가 명시
- **대화 전달** — 인물이 다른 인물에게 설명
- **암시** — 독자만 눈치채는 복선
- **폭로 장면** — 드라마틱하게 밝혀짐
- **인물 추론** — 인물이 스스로 알아냄

## 정보 은폐 전략
- 독자보다 인물이 더 많이 아는 구간 (공포/조마조마):
- 인물보다 독자가 더 많이 아는 구간 (아이러니/연민):
- 독자도 인물도 모르는 구간 (미스터리):
