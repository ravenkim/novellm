# 분기 트리 개요

> 소설의 핵심 선택지와 결과 구조.
> 작가가 채택한 경로를 표시하고, 버려진 가지는 참고용으로 유지.

## 전체 분기도

```
초기 상태
│
├── [분기점 1 — 화: ]
│     ├── 선택 A: 
│     │     └── 결과: → 분기점 2로
│     └── 선택 B: 
│           └── 결과: → 분기점 3으로
│
└── ...
```

## 핵심 분기점 목록
| 화 | 분기점 설명 | 선택 A | 선택 B | 채택 경로 | 사건 ID |
|----|-----------|--------|--------|---------|--------|
|    |           |        |        | A / B   |        |

## 버려진 가지 메모
> 채택하지 않은 경로. 서브플롯이나 후속 작에 활용 가능.
