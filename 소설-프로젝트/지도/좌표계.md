# 좌표계 정의

## 그리드 기준
- **열 (X축)**: A ~ Z (서쪽 → 동쪽)
- **행 (Y축)**: 1 ~ 99 (북쪽 → 남쪽)
- **셀 표기**: 열+행 → `C4`, `G12`
- **1칸 = ** (예: 100km / 도보 3일 / 말 1일)  ← 세계 규모에 맞게 설정

## 정밀 좌표 (소수점)
큰 도시 내부나 던전 등 세밀한 위치는 `C4.2` 형태로 표기  
(C열 4행 안에서 세로 2/10 지점)

## 기준점 (원점)
- 원점 `A1` = 지도 북서쪽 끝
- 기준 도시: `___` (좌표: ___)  ← 이야기의 중심지

## 방위
```
        북 (행 번호 작아짐)
         ↑
서 ← A B C D ... → 동
         ↓
        남 (행 번호 커짐)
```
